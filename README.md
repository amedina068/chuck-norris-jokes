# Chuck Norris Jokes

Create Chuck Norris Jokes in your app

## Installation

```bash
require composer amedina/chuck-norris-jokes
```

## Usage

```php
use Amedina\ChuckNorrisJokes\JokeFactory;

$jokes = new JokeFactory;
$joke = $jokes->getRandomJoke();
```
